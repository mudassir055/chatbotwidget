üöÄ Quick Integration Guide

Follow these simple steps to integrate the chatbot widget into any website:

Step 1: Deploy the Widget

Choose one of these deployment options:

Option A: Vercel (Recommended)

# Install Vercel CLI npm install -g vercel # Deploy from this directory vercel # Follow the prompts # Your widget will be available at: https://your-project.vercel.app/embed.js

Option B: GitHub Pages

1. Push this code to a GitHub repository 2. Go to Settings > Pages in your repo 3. Enable GitHub Pages from main branch 4. Your widget will be at: https://username.github.io/repo-name/embed.js

Option C: Netlify Drop

1. Go to https://netlify.com/drop 2. Drag and drop this entire folder 3. Get instant deployment URL 4. Your widget will be at: https://random-name.netlify.app/embed.js

Step 2: Configure Your Webhook

Update the webhook URL in the integration code below to point to your backend endpoint.

Step 3: Add to Any Website

Copy and paste this code into any HTML page:

<!-- Basic Integration --> <script> window.ChatbotConfig = { webhookUrl: 'https://your-webhook-endpoint.com/webhook', // ‚Üê Change this! title: 'Support Assistant', position: 'bottom-right' }; </script> <script src="https://your-deployed-url.com/embed.js"></script> <!-- ‚Üê Change this! -->

Step 4: Test Integration

Create a simple test page:
