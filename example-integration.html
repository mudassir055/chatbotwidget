How It Works

The chatbot widget operates by sending user messages to your configured webhook endpoint. Here's the flow:

  1. User Input: User types a message or speaks using voice input
  2. Webhook Call: Widget sends the message to your webhook URL
  3. Processing: Your backend processes the message (AI, database lookup, etc.)
  4. Response: Your webhook returns a JSON response with the bot's reply
  5. Display: Widget shows the response and optionally speaks it aloud
