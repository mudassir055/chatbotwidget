Webhook Integration

The widget sends POST requests to your webhook with this format:

{
    "message": "User's message",
    "timestamp": "2024-01-01T12:00:00.000Z",
    "type": "text"
}

Your webhook should respond with:

{
    "response": "Bot's reply message"
}
